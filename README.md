#  Portafolio Web - Proyecto Final

Este repositorio contiene el código fuente de mi portafolio personal. Fue creado como proyecto final para demostrar la correcta estructuración de una página web desde cero, integrando los que aprendi.

##  Explicacion
Se me ocurrio,hacer la estructura de un web sencilla pero con el fin de ser una pagina que use como portafolio sobre mi trabajos.Tiene una distribucion simple,un menu secillo,una muetras de mi trabajo, contacto y una consulta personalizada que funciona con un formulario.

##  Demo en Vivo
Visita la pagina en vivo aqui:
 https://github.com/akisan862-crypto/proyecto-final-ash.git

##  Tecnologías Utilizadas
* **HTML5 Semántico:** Uso de etiquetas estructurales como `<header>`, `<main>`, `<section>`, `<article>`, `<nav>` y `<footer>`.
* **Git y GitHub:** Para el control de versiones local y el alojamiento del proyecto.
